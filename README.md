## Work

<br/>

**AURA** &mdash; Autonomous AI Ecosystem &nbsp;·&nbsp; *Active*

Multi-agent AI platform built for full edge deployment. Local inference via Ollama, real-time orchestration with LangChain, persistent memory, voice interaction with Whisper AI, and an immersive Next.js interface over WebSockets. Zero cloud dependency by design.

`Local LLM` `Ollama` `LangChain` `Whisper AI` `Next.js` `WebSockets` `Python`

<br/>

**SHADOWGUARD** &mdash; AI Security & DLP Engine &nbsp;·&nbsp; *Active*

Enterprise architecture preventing unauthorized LLM data exfiltration. Real-time packet inspection, behavioral anomaly detection, and policy enforcement at the inference layer. Designed for environments where AI adoption outpaces security posture.

`DLP Systems` `Deep Packet Inspection` `ML Classifiers` `Python`

<br/>

**NIDS CORE** &mdash; Network Intrusion Detection &nbsp;·&nbsp; *Active*

ML-powered intrusion detection using packet-level analytics and multi-class anomaly classification. Built for real-time threat intelligence and low-latency alerting across production network segments.

`Scikit-Learn` `Packet Analytics` `Anomaly Detection` `Python`

<br/>

**STP SYSTEM** &mdash; Autonomous Cleaning Robot &nbsp;·&nbsp; *Shipped*

Sewage treatment robot engineered under ₹3,000 INR. Intelligent waste classification, embedded motor control, and sensor fusion on severely constrained hardware. Proof that good engineering is mostly about constraints.

`ESP32` `Arduino` `Sensor Fusion` `Edge AI` `C++`

<br/>

**SUGAR AI** &mdash; Offline Multimodal Assistant &nbsp;·&nbsp; *Active*

Fully offline multimodal assistant with local vision, speech recognition, and language understanding. No API calls. No cloud. Runs on edge hardware where connectivity is either unavailable or untrusted.

`TensorFlow` `Whisper` `Offline LLM` `Python`

<br/>

**Embedded / IoT Systems** &mdash; Ongoing

Sensor fusion arrays, IoT telemetry pipelines, and edge-AI inference on microcontroller-class hardware. The kind of systems that run 24/7 in places where you can't ssh in to debug.

`ESP32` `Arduino` `C` `C++` `IoT` `Edge Computing`
